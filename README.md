# 🚗 Parking Management System

## 📌 Project Overview
The Parking Management System is a Python-based mini project that helps manage vehicle parking efficiently. It records vehicle entry and exit, checks parking slot availability, and calculates parking fees based on parking duration.

## ✨ Features
- Vehicle Entry
- Vehicle Exit
- Parking Slot Availability
- Parking Fee Calculation
- View Parked Vehicles
- Simple Menu-Driven Interface

## 🛠️ Technologies Used
- Python
- datetime module

## 🌍 Real-World Applications
- Shopping Malls
- Hospitals
- Railway Stations
- Airports
- Office Buildings
- Residential Parking

## 🚀 How to Run
1. Download or clone the repository.
2. Open the project in VS Code.
3. Run:
```bash
python parking_management.py
```
